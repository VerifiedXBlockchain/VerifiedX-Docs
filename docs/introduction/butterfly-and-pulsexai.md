# Butterfly & PulseXAI

**Butterfly** is a payments and financial access application built on the VerifiedX network. It enables instant, near-zero-cost global transfers using any supported currency — no downloads, no personal data storage, and no middlemen. Users send and receive funds as simply as sending a message, with self-custodial ownership from the moment they log in. Butterfly also provides built-in yield capabilities, allowing balances to work passively with no holding periods or penalties.

**PulseXAI** extends the VerifiedX ecosystem into programmable intelligence. It enables users and institutions to deploy autonomous on-chain agents, execute financial strategies programmatically, and capture value from machine-driven economic activity — all secured by the same infrastructure that powers vBTC and the broader VerifiedX network.
