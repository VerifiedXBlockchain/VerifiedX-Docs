# VFX Ecosystem

The VFX Ecosystem Console is an interactive visualization of the entire VerifiedX architecture — from Bitcoin custody and vBTC issuance through to DeFi vaults, smart contracts, payments, and agentic intelligence. Explore how each layer connects across SwitchBlade, Butterfly, PulseXAI, privacy, self-custodial vaults, and more, with real-time network data alongside.
